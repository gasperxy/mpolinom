Instructions

Search/contribution example of M-polynomial:
(2p+6) x^2 y^2 + (8p+8q−4) x^2 y^3 + (15pq−10p+2q−1) x^3 y^3

Contribute new M-polynomial

IMPORTANT: All M-polynomials must be functions of variables x and y.

  • Log in (contact dr. Sandi Klavžar to gain username and password)
  • Choose Add and complete the form
  • Note: structure name must be unique in the database. If the structure name already exsits, and your M-polynomial differs from the one in the database, please contact dr. Sandi Klavžar.

Add comment

  • Log in (contact dr. Sandi Klavžar to gain username and password)
  • Choose Change
  • Select M-polynomial you wish to change. You can use search box but note that the search is simplified and might not give you the best results when searching by M-polynomials. Instead we recommend search by structure name or id.
  • To add comment, link, reference or keyword simply write comma between existing comments and new comment
